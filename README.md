## 🎮 游戏简介

《AFO：结束？开始？》是一款基于浏览器的高中文化课学习模拟游戏。玩家将扮演一位从信息学竞赛（OI）退役的选手，在52周的时间内，通过策略性的选择平衡学习、休息、社交、经济等多维度状态，最终在高考中取得理想成绩。

> "从代码到考卷，这是一场关于成长与救赎的史诗旅程"

## ✨ 核心特色

### 🎯 策略性游戏机制
- **每周3次操作限制**：必须在有限资源下做出最优决策
- **多维度属性系统**：精力、压力、健康、心理、经济、人际关系相互影响
- **真实的高中生活模拟**：课程表、作业、模考、假期等完整系统

### 📚 丰富的游戏内容
- **52周完整学年**：从AFO第一天到高考结束
- **随机事件**：学习、关系、健康、经济、特殊事件五大类别
- **故事剧情**：主线故事线，体验从迷茫到成长的完整心路历程
- **成就系统**：基础、状态、学科、关系、活动、特殊、结局七大类成就
- **不同结局**：根据最终表现解锁不同人生道路

### 🎨 精美的界面设计
- **现代化玻璃拟态UI**：美观的渐变背景和毛玻璃效果
- **响应式布局**：适配桌面和移动设备
- **实时状态监控**：直观的进度条和统计面板
- **事件日志系统**：记录所有重要事件和选择

## 🕹️ 游戏玩法

### 核心状态管理
- **精力值**：学习、活动消耗，休息恢复
- **压力值**：考试、作业增加，休息、活动减少
- **健康值**：熬夜、压力降低，锻炼、休息提高
- **心理值**：心理咨询、积极事件提高
- **经济值**：兼职、父母支持获得，购买资料消耗

### 学科能力提升
- **六大学科**：语文、数学、英语、物理、化学、生物
- **竞赛背景加成**：数学和物理有额外学习效率
- **心理状态影响**：心理状态直接影响学习效率

### 人际关系系统
- **8个角色**：班主任、各科老师、同学、父母、暗恋对象
- **关系影响**：关系好坏影响学习支持和心理状态
- **隐藏剧情**：达到特定关系值解锁特殊事件

## 🎮 操作指南

### 主要操作按钮
1. **📚 学习** - 选择科目重点学习
2. **💤 休息** - 选择休息方式恢复状态
3. **🏃 锻炼** - 体育锻炼释放压力
4. **👥 社交** - 与同学交流改善关系
5. **💭 回忆** - 回忆竞赛往事，影响心理状态
6. **📝 模考** - 参加模拟考试检验学习成果
7. **✏️ 作业** - 完成作业巩固知识
8. **🎉 活动** - 参加课外活动丰富经历
9. **💰 经济** - 管理零花钱和支出
10. **🌙 结束本周** - 推进时间到下一周

### 键盘快捷键
- `1` - 学习
- `2` - 休息
- `3` - 锻炼
- `4` - 社交
- `5` - 回忆
- `6` - 模考
- `7` - 作业
- `8` - 活动
- `9` - 经济
- `0` - 结束本周
- `Shift` - 显示快捷键提示
- `Esc` - 关闭所有弹窗

## 🛠️ 技术细节

### 文件结构
```
AFO-Simulator/
├── index.html          # 主游戏文件（包含所有HTML、CSS、JS）
├── README.md           # 项目说明文档
└── 其他文件
```

### 技术栈
- **HTML5**：游戏结构
- **CSS3**：现代化UI设计，玻璃拟态效果
- **JavaScript**：游戏逻辑和交互
- **响应式设计**：适配不同设备

### 游戏状态管理
游戏使用单一`gameState`对象管理所有状态，包括：
- 时间系统（周、日、季节）
- 核心状态（精力、压力、健康等）
- 学科能力（六科分数）
- 人际关系（8个角色）
- 游戏进度（成就、事件、统计）
